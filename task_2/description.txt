2. Кэш данных об облигациях

Дана функция, которая получает из API данные о финансовых показателях облигаций за заданную дату по массиву идентификаторов облигаций (ISIN):

const getBondsData = async ({date, isins}) => { const result = await http.post({ url: `/bonds/${date}`, body: isins });

return result; };

Пример вызова функции:

getBondsData({ date: '20180120', isins: ['XS0971721963', 'RU000A0JU4L3'] });

Результат:

[{

isin: 'XS0971721963', data: {...} }, { isin: 'RU000A0JU4L3', data: {...} }]

Задача

Изменить код функции, реализовав кэш на стороне клиента.

Ожидаемое решение

● программа на языке JavaScript в виде одного исходного файла или сниппета на любом открытом веб-хостинге, без использования сторонних библиотек,

● вычислительная сложность алгоритма и оценка необходимой памяти для его выполнения.
